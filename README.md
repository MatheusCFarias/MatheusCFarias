## 📊 Metodologia de Trabalho

Sigo uma abordagem estruturada em todos os projetos:

1. **Entendimento do Negócio** - Definir perguntas-chave e KPIs relevantes
2. **Coleta de Dados** - Extração via SQL, APIs ou arquivos
3. **Limpeza & Transformação** - Tratamento de missing values, outliers e normalização
4. **Análise Exploratória** - Identificação de padrões, correlações e anomalias
5. **Visualização** - Criação de dashboards intuitivos e storytelling
6. **Insights & Recomendações** - Tradução de dados em ações estratégicas
